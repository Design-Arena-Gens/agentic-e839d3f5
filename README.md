# Agentic Website Maintainer

Autonomous control room for keeping your marketing site healthy, searchable, and fast. The `agentic/` app bundles an interactive dashboard, directive console, and maintenance pipeline tailored for rapid deployment on Vercel.

## Quickstart

```bash
cd agentic
npm install
npm run dev
```

Visit `http://localhost:3000` to fly the agent.

## Highlights

- Continuous health telemetry and actionable status cards.
- Natural-language directive console that turns prompts into structured work.
- Kanban-style maintenance pipeline with instant status transitions.
- Real-time activity journal documenting the agent's decisions.

## Deployment

The project is optimized for Vercel. Build locally with `npm run build`, then deploy using `vercel deploy --prod`.
